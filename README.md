## 📖 Overview
A local utility tool developed intelligently in **Python** to manage, sort, and process large collections of images seamlessly. Excellent for dataset preparation in AI/ML structures or just broad file administration.

## ✨ Elements
- **Efficient Processing**: Leverages modern Python libraries to perform quick, batched transforms.
- **Automated Directory Tracking**: Scans multiple layers of sub-directories cleanly categorizing image payloads based on their metadata and characteristics.
- **Data Science Pre-Requisite**: Perfect pipeline element for preparing pristine, organized inputs prior to training model feeds.

## 🛠️ Usage
Download the code module to any environment having Python execution rights:
```bash
git clone https://github.com/Imranxhah/image_manager.git
cd image_manager
# Adjust your target directories inside the main execution file before running.
python run.py
```
